## Overview

LibGDX ships without higher-level tools for drawing and manipulating polygons. To unblock our workflow, I ported Cornell's CUGL geometry module into LibGDX and wrapped it with an interactive demo suite. The toolkit now covers polyline authoring, extrusion, smoothing, and spline generation directly in engine.

## Key Classes

- **Path2:** Core polyline container with winding checks, containment tests, scale/translate utilities, and slice operations.
- **Poly2:** Triangulated polygon representation with boundary detection, mesh adjacency, and transformation helpers.
- **Spline2 & SplinePather:** Author cubic Bézier splines, then convert them through SplinePather → Path2 → SimpleExtruder → Poly2.
- **PathFactory & PolyFactory:** Convenience constructors for common primitives with configurable dimensions and offsets.
- **PathSmoother & SimpleExtruder:** Utility classes that smooth user input and create solid meshes from open paths.
